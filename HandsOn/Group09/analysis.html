Data Set Analysis

Information on the concessions registered on the assets of the Inventory of Public Areas and Green Zones, detailing type of concession, location and name of applicant.

  • Location: districts and particular areas of Madrid.
  • Type of concession: Types such as lease, permit, etc.
  • Identificator of the particular concession: numbers between 12000 and 16000.
  • Identificator of the applicant: name of the commerce, association, public area, etc.

Licensing Analysis

The dataset is provided by the Comunidad de Madrid through its open data portal. The data is publicly available for use and likely adheres to open data licensing, such as Creative Commons CC BY 4.0, but the exact terms should be verified on the source website.

Resource Naming Strategy

The URI structure follows the best practices in Linked Data. We propose the following structure for naming resources:

  • Base URI: http://smartcity.linkeddata.es/concesiones_zonas_verdes/
    • Ontology: http://smartcity.linkeddata.es/concesiones_zonas_verdes/ontologia/...
      • Class: .../{class}
      • Property: .../{property}
    • Data: http://smartcity.linkeddata.es/concesiones_zonas_verdes/recursos/...
      • Resource: ...recursos/{class}/{id}

For example:

  • Concession URI example: http://smartcity.linkeddata.es/concesiones_zonas_verdes/concesion/12000
  • District URI example: http://smartcity.linkeddata.es/concesiones_zonas_verdes/distrito/Retiro

This resource naming strategy ensures that every entity (concession, district, commerce, ...) has a unique identifier and is accessible following Linked Data principles.
