Data Set Analysis

The dataset contains information about trees in 19 parks in Madrid, detailing the state of the trees (young, mature, old, etc.). The key statistics for the dataset are as follows:

  • Total Parks: 19
  • Total Trees: 687,777
  • Average number of newly planted trees: 65.87 per park, with a maximum of 352 trees in a single park.
  • Average number of young trees: 135.3 per park, with a maximum of 745.
  • Average number of mature trees: 145.97 per park, with a maximum of 859.
  • Average number of old trees: 167.19 per park, with a maximum of 934.
  • Other trees category: 2.63 trees on average per park.

  • Base URI: http://smartcity.linkeddata.es/arbolado/
  • Tree URIs: http://smartcity.linkeddata.es/arbolado/arboles/{tree_id}

For example:

  • Tree URI example: http://smartcity.linkeddata.es/arbolado/arboles/12345
