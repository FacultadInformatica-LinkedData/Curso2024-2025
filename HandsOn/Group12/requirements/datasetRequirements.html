Dataset Requirements

R1. Data are in one of the selected smart city domains.

This dataset is about the traffic light's location on the area of Madrid.

R2. Data are available as a CSV file.

This dataset is available in a CSV file

R3. Data have an open license so they can be published.

This dataset has an open license: URL

R4. Data can be easily linked with generic real-world entities (e.g., locations).

This datasset contains the UTM (Universal Transverse Mercator) of each traffic light, columns named utm_x and utm_y.

R5. Documentation exists for the data (optional).

There is any additional documentation available on the site for this dataset.

R6. Data can come from multiple data sources (optional).

This dataset only comes from one source.
