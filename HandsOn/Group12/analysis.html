Analysis

Data Set Analyse.

This dataset is about the traffic light's location on the area of Madrid. It contains 56709 traffic lights distributed on Madrid. The data set contains information such as elemet id, element type, element location and further information:

Column name Data type Format Description
tipo_elem String SEMAFORO x/xxx [type] Type of traffic light, where x indicates number, type contains{PEATON, BICI, PEATON - BICI}.
distrito Integer x District of the location of the traffic light, where x indicates number.
id Integer x Unique id for the traffic light, where x indicates number
id_cruce Integer x Non-unique id for the traffic light intersection, where x indicates number. Elements with same id_cruce means they are on the same intersection.
fecha_inst Time dd/mm/yyyy hh:mm Instalation date in format day/month/year hour:minute
utm_x String x.x.x.x.x X-coordinate for the Universal Transverse Mercator, where x indicates any number from 0 to 999.
utm_y String x.x.x.x.x.x Y-coordinate for the Universal Transverse Mercator, where x indicates any number from 0 to 999.
longitud String x.x.x.x.x Longitude for the Geographic Coordinate System, where x indicates any number from 0 to 999.
latitud String x.x.x.x.x.x Latitude for the Geographic Coordinate System, where x indicates any number from 0 to 999.
