Introduction

Dataset relacionado con las carreteras con mayor conduccion temeraria de la ciudad de Madrid Traffic Accident Dataset The data will be organized by:

  • Mes: Mes del accidente.
  • Localizacion: Carretera o Autopista del accidente.
  • Distrito: Distrito de Madrid.
  • Tipo de Accidente: Tipo de accidente, ya sea por otros vehiculos o obstaculos.
  • Tipo de Vehiculo: Vehiculo involucrado.
  • Drogas/Alcohol: Si el conductor habria tomado drogas o alcohol

Namespace

El Namespace se diseñará para distinguir los términos ontológicos y las instancias de la ontología.

  • Para los términos ontológicos (clases y propiedades)

    Ejemplo de URI para la clase 'Accidente': https://websemantica.net/Group22/ontology/AccidentesMad#Accidentes

    Ejemplo de URI para la propiedad de los accidentes 'tieneInvolucrados': https://websemantica.net/Group22/ontology/AccidentesMad#tieneInvolucrados

  • Para las instancias individuales (como accidentes específicas, involucrados, y lesionados):

    Ejemplo de URI para un accidente específica con ID '2024S000001': https://websemantica.net/Group22/resource/Accidente/2024S000001

    Ejemplo de URI para un involucrado específico con código '2024S000001_1': https://websemantica.net/Group22/resource/Involucrado/2024S000001_1
